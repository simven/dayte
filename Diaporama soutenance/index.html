Présentation de la mission

  • Réalisation d’une application de rencontre (web et mobile)

  • Alternative aux sites de rencontre existants

  • Application : met en relation deux individus selon un questionnaire (unique)

  • Ces individus n’ont aucun pouvoir de décision sur le choix de leur partenaire

  • Aucun contact virtuel avant la première rencontre physique

Environnement et outils utilisés

  • Framework utilisé : Angular, avec NativeScript Schematics pour la programmation mobile

  • SQL : PostgreSQL

  • API REST : PostgREST

Utilisation d’outils collaboratifs tels que LifeSize, Discord et GitLab
